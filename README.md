# M14service

Marketing site for M14service — cleaning services (carpet, hard floor, upholstery, factory & warehouse, high level, daily / deep / move-in–out).

## Stack
- Next.js 16 (App Router, Webpack)
- React 19
- Tailwind CSS v4
- Lenis (smooth scroll)
- TypeScript

## Develop
```bash
npm install
npm run dev
```
Open http://localhost:3000.

## Add images
Drop files into `public/images/`. Reference them in components as `/images/your-file.jpg`.
